# Signal Types Classification

Решение задачи кластеризации сигналов сцинтилляционного детектора
на три класса: гамма-кванты, нейтроны и аномальные сигналы.

## Результат на Kaggle
Лучший score: **0.823**

## Что смотреть

**Сессионное_задание.ipynb** - основной ноутбук с полным решением:
- EDA и визуализация сигналов
- Извлечение признаков из временных рядов
- Сравнение моделей KMeans, GaussianMixture, DBSCAN
- Подбор гиперпараметров
- Финальное предсказание

**Снимок.PNG** - скриншот таблицы лидеров

## Как запустить

1. Загрузи файл `Run200_Wave_0_1.txt` в Google Drive
2. Открой ноутбук в Google Colab
3. Запусти все ячейки по порядку

## Процесс улучшения результата

| Версия | Изменение | Score |
|--------|-----------|-------|
| v1 | KMeans, базовые признаки | 0.755 |
| v2 | Подбор окна хвоста (offset=7) | 0.759 |
| v3 | Добавлен признак ratio2 (два окна интегрирования) | 0.767 |
| v5 | Замена KMeans на GaussianMixture tied | 0.822 |
| v7 | Добавлен признак centroid_time | 0.823 |

Главный скачок дала замена KMeans на GaussianMixture с
covariance_type='tied' эта модель лучше справляется с
вытянутыми кластерами неодинакового размера.

## Финальная модель

GaussianMixture, covariance_type='tied', n_components=3

Признаки: psd_ratio, ratio2, peak_amplitude_log, decay_time,
tail_to_peak_log, baseline_std, centroid_time
